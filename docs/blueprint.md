# **App Name**: LakminaSecure

## Core Features:

- About Me Section: Detailed personal and professional overview of Ravindu Lakmina, highlighting his experience and philosophy in cybersecurity.
- Cybersecurity Skills Showcase: Interactive listing of Ravindu's skills with proficiency levels indicated visually. Categories might include penetration testing, incident response, and compliance.
- Projects Showcase: Gallery or list of completed projects with descriptions, technologies used, and outcomes, demonstrating practical application of skills.
- Resume Download: Allow users to easily download Ravindu Lakmina's resume in PDF format.
- Contact Form: Allow potential clients and employers to quickly get in touch using a straightforward form. Direct submissions to Ravindu's professional email.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to represent cybersecurity themes and provide a vibrant highlight on the dark background.
- Background color: Dark gray (#333333), almost black, providing a professional, serious backdrop.
- Accent color: Slightly lighter Gray (#555555) for subtle contrast and improved readability without distracting from the electric blue highlights.
- Font pairing: 'Inter' (sans-serif) for body text and 'Space Grotesk' (sans-serif) for headlines, ensuring readability and a modern feel. Note: currently only Google Fonts are supported.
- Use minimalist, neon-style icons for skills and project categories.
- A clean, single-page layout with clear sections and easy navigation to ensure a user-friendly experience.
- Subtle animations on page load and when hovering over projects/skills.
